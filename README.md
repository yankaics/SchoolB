##校园宝是什么?
方便自己学院任何人更便捷的体验校园

账号（学号or工号or账号） 密码（身份证后6位）

目前校园宝搭建在学院的服务器上-也就是必须登录学院的校园网才能进行访问

校园宝目前功能不多，很多方便的功能已在计划当中。
***
##校园宝有哪些功能？

* 学院‘公物报修’。
    *  填写学院基本信息。
    *  选择需要报修的物件。
    *  核对信息，填写损坏描述-提交。
* 公物报修成功之后进入报修查询-查看报修情况。
* 默认密码推荐尽快修改密码。
计划中的功能
* 宿舍水电费查询-缴费
* 学生信息的查看
* 。。。。。。。。
